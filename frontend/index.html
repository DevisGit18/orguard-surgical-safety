Dataset Reference — CholecSeg8k
CholecSeg8k — Hong et al., 2020. 8,080 frames from 17 laparoscopic cholecystectomy videos with pixel-level segmentation masks for 13 classes: grasper, hook, clipper, irrigator, specimen-bag, and hepatocystic triangle tissues.

Expected video format: Laparoscopic camera feed, 854×480 px, 25 fps, MP4.

Flask endpoint spec: POST http://localhost:5001/detect Content-Type: multipart/form-data · Body: { video: <file> } Response: { detections: [{ label: string, confidence: float, frame: int }], summary: { [label]: count } }
